# Word Stash
[![Certified Shovelware](https://justin.searls.co/img/shovelware.svg)](https://justin.searls.co/shovelware/)

Save your articles for later! (without worrying about them being deleted)

(also LLM enhanced!)

## Local Setup
- `brew install ollama`
- `ollama pull llama3.2`
- `ollama serve`
